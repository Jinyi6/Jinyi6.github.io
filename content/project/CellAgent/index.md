CellAgent is presented here as a usable research system rather than only a paper artifact. The live interface shows how researchers can move from a natural-language request to a structured single-cell analysis workflow in the browser.

<!--more-->

Compared with the publication page, this detail view emphasizes the product-facing layer: how planning, tool orchestration, and domain-specific analysis are surfaced in an accessible workflow for researchers who do not want to stitch together custom scripts by hand.

If you want the method, evaluation, and technical framing, start with the paper. If you want to understand the interaction flow and deployment experience, start with the live system.
