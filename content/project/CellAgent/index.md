An LLM-driven multi-agent system that lets researchers run end-to-end single-cell analysis through natural language while preserving high-quality scientific outputs.

<!--more-->

This page focuses on the system and deployment side of CellAgent. For the paper version with method details and results, see [the related publication](/publication/xiao-2024-cellagent/).

CellAgent packages planning, tool use, and domain-specific analysis steps into a workflow that researchers can access through a browser instead of custom scripting.
